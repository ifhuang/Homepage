Introduction to Database Systems (Feb. 2014 - Jun. 2014)


This is the first course of database systems for undergraduates in computer science. It offers students an introduction to the theoretical foundation and implementing techniques of relational database management systems, the design and implementation skills of database applications. In particular, the course will cover

1) The foundation of Relational databases, including ER (entity-relationship) model, relational model, SQL, query processing and optimization, and relational database design methods.

2) The core techniques of Relational Database Management Systems (RDBMS), including data organization, index structures, transaction management, concurrency control and database recovery.

3) Some advanced topics of databases systems, including XML data management and database application development.
